# uploader-with-input-file for Deno

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

ファイル選択拡張タグ [input-file](https://github.com/code4fukui/input-file) を使ったアップローダー

A simple Deno-based file uploader that uses the [`<input-file>`](https://github.com/code4fukui/input-file) web component for file selection and [`PubkeyUser`](https://github.com/code4fukui/PubkeyUser) for authentication. This project serves as a practical example of combining these components to create a secure, client-side upload system.

## Features

- **Public Key Authentication:** Secures the upload endpoint using `PubkeyUser`, allowing only pre-approved users to upload files.
- **Custom File Input:** Utilizes the `<input-file>` web component for a modern file selection experience.
- **Server-Side Storage:** Uploaded files are stored on the server in a `data/` directory (ignored by `.gitignore`).
- **Unique File IDs:** Each uploaded file is assigned a Time-based ID ([TID](https://github.com/code4fukui/TID)) for its filename, preventing collisions.

## How It Works

1.  When you open the application in your browser, a new public/private key pair is generated by `PubkeyUser` and stored in your browser's local storage. Your public key is displayed in the top input field.
2.  To gain upload access, an administrator must add this public key to the `allowedUsers.json` file on the server.
3.  When you select files using the `<input-file>` component, they are automatically uploaded to the server.
4.  The server validates your public key against the `allowedUsers.json` list before accepting and saving the file.
5.  Each successfully uploaded file is saved in the `data/` directory with a unique filename based on its TID and original extension.

## Getting Started

### 1. Prerequisites

Ensure you have [Deno](https://deno.com/) installed.

### 2. Run the Server

Clone the repository and run the following command in the project directory:

```sh
deno serve -A --port 8010 server.js
```

The server will start on `http://localhost:8010`.

### 3. Authorize a User

1.  Open `http://localhost:8010` in your web browser.
2.  Copy the public key string that appears in the first input box.
3.  Open the `allowedUsers.json` file in the project directory.
4.  Add your copied public key as a new string to the JSON array and save the file.

    ```json
    [
      "YOUR_COPIED_PUBLIC_KEY_HERE",
      "VN7R_CC8U_L4K9_RWMN_VRVJ_TZN5_CZ0F_3KQF_UQHP_QVNZ_2DF5_CGCM_65H0"
    ]
    ```

### 4. Upload Files

Refresh the page in your browser. You can now select files using the file input component, and they will be uploaded to the `data/` directory on the server.
